### 🎉 依赖安装

后端相关依赖安装和功能说明：

```bash
# cobra：命令行工具
go get -u github.com/spf13/cobra

# viper：配置文件解析
go get -u github.com/spf13/viper

# gin：Web 框架
go get -u github.com/gin-gonic/gin

# go-pretty：打印表格
go get -u github.com/jedib0t/go-pretty/v6
```
